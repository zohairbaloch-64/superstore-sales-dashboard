# 📊 Superstore Sales Dashboard

## 🚀 Live Demo
[Click Here to View Dashboard]https://superstore-sales-dashboard-z43.streamlit.app/

---

## 🎯 Project Objective
Analyze Superstore sales data to uncover business insights and build an interactive dashboard for decision-making.

---

## 📁 Dataset
- Superstore Orders Dataset
- Includes sales, profit, customers, products, and regions

---

## 🛠 Tech Stack
- Python (Pandas, NumPy)
- Plotly (Visualization)
- Streamlit (Dashboard)

---

## 📊 Features
- KPI Metrics (Sales, Profit, Orders, Margin)
- Interactive Filters (Region, Category, Date)
- Sales Trend Analysis
- Profitability Analysis
- Top Customers & Products
- Loss-Making Insights

---

## 🧠 Key Insights
- High sales do not always mean high profit
- Some categories consistently generate losses
- A small group of customers drives major revenue
- Discounts significantly impact profitability

---

## 💡 Business Recommendations
- Optimize discount strategies
- Focus on high-margin products
- Improve loss-making categories
- Strengthen top customer retention

---

## ▶️ How to Run Locally


pip install -r requirements.txt
streamlit run app.py
